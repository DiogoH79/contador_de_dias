CONTROLE DE BICOS

Como usar no celular:
1. Abra o arquivo index.html no Chrome.
2. Toque nos três pontinhos do Chrome.
3. Escolha "Adicionar à tela inicial" ou "Instalar app".
4. O app fica salvo no celular e os dados ficam no próprio aparelho.

Recursos:
- Calendário mensal.
- Agendar trabalho com dia, descrição, horário, local e valor.
- Marcar como recebido ou a receber.
- Financeiro mensal.
- Backup/exportação dos dados.
